# Doom's Markers

Doom's Markers is a mod that adds similar functionality to map mods "Waypoints" without needing a whole map mod as well.

## Features
- [x] Death Markers
- [x] Show distance to Markers
- [x] Customise Markers with Dye and items
- [ ] Share Markers
- [ ] Integrate map markers for treasure maps etc

## Controls

Everything revolves around the `Use` key. It defaults to the grave key `

Pressing the `Use` key and releasing it without doing anything will place down a new Marker.

Holding the `Use` key and left clicking, or punching, a Marker will delete it.

Holding the `Use` key and middle clicking with the mouse on a Marker while holding an item will change the Marker to display with that item.

Holding the `Use` key and scrolling with the wheel will scroll through the built in Marker icons.

Holding the `Use` key and right clicking on a Marker while holding a Dye with dye the Marker that colour.
